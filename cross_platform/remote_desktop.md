Platforms which not require account.

  Signal messenger

- 2

  http://web.rustdesk.com/

- 3

  http://whereby.com

  - account can be created with fake email. no OTP needed.
  - it based on browser's desktop-share feature. Therefore you can see remote desktop but you can not control the mouse or keyboard. (tested on Firefox)

  TeamViewer (inside virtual machine)

  https://www.dwservice.net/ (if you are server prefer inside virtual machine)

# Remmina
"Remmina" works only on Linux (via Flatpak or snap). It is a GUI client for RDP(Microsoft), SSH, VNC.
